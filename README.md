# gary-legal

Privacy Policy

Service: Gary Botman Personal Assistant
Last updated: June 7, 2026

Overview

Gary Botman Personal Assistant is a private, personal AI assistant operated solely for the use of its owner. This service sends automated SMS reminders and notifications exclusively to the account owner's mobile phone number.

Information Collected

The only personal information collected and used by this service is:

The account owner's mobile phone number, used solely for message delivery
Message content necessary to deliver reminders and notifications
How Information Is Used

Information is used exclusively to deliver automated reminders and notifications (such as water intake reminders and market intelligence alerts) to the account owner. No information is used for any other purpose.

Data Sharing

No personal information is shared with third parties, sold, or used for marketing purposes of any kind. This is a private, single-user service.

Data Retention

Message logs are retained locally on the owner's private server and are not transmitted to or stored by any third party beyond the SMS carrier used for delivery.
